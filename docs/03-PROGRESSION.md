## 1. 레벨 시스템 (1-50)

### 레벨업 곡선
```
nextXp = floor(nextXp × 1.22 + 35)

LV 1→2:  140 XP (적 14마리)
LV 5→6:  290 XP
LV 10→11: 580 XP
LV 20→21: 1,400 XP
LV 30→31: 3,200 XP
LV 40→41: 7,500 XP
LV 49→50: 17,000 XP
```

## 2. 무기 강화 (Lv.0 → Lv.3)

### 강화 경로
```
[무기 획득] → Lv.0 (기본)
[같은 무기 재획득] → Lv.1 (피해 +15%, 탄약 +25%)
[같은 무기 재획득] → Lv.2 (피해 +30%, 탄약 +50%)
[같은 무기 재획득] → Lv.3 (피해 +45%, 탄약 +75%) ★MAX
```

### Lv.3 특수 효과
| 무기 | Lv.3 보너스 |
|------|------------|
| HMG | 관통 (적 2마리까지) |
| Rocket | 연쇄 폭발 (반경 100 → 130) |
| Arc | 체인 +2 (4→6마리) |
| Laser | 2발 동시 발사 |
| Railgun | 3발 산탄 (spread 0.08) |

### 강화 시각 피드백
- Lv.1: 무기 이름 옆 ★ (노란색)
- Lv.2: ★★ + 총알 글로우 증가
- Lv.3: ★★★ + **무기 오라** (플레이어 손에서 빛남) + 전용 사운드

## 3. 무기 합성

### 합성 조건
- 2슬롯 모두 **Lv.3**
- Q+L 동시 입력
- 합성 후: Lv.3 무기 2개 소모 → 합성 무기 1개 생성

### 합성 무기 스탯

| 이름 | 조합 | 피해 | 연사 | 탄약 | 특수 |
|------|------|------|------|------|------|
| 플라즈마 스톰 | HMG+Laser | 35 | 0.04 | 40 | 관통 + trail |
| 썬더 크래시 | Rocket+Arc | 90 | 0.6 | 15 | 폭발 + 4체인 |
| 개틀링 레일 | Railgun+HMG | 50 | 0.15 | 30 | 관통 + 3발 |
| 퓨전 블래스터 | 기타 | 45 | 0.12 | 35 | 폭발 반경 50 |

### 합성 도감
- 4종 발견 현황: ⬜⬜⬜⬜ → ⬛⬛⬜⬜ → ⬛⬛⬛⬛
- **전부 발견 시**: "OMEGA BLASTER" 해금 (피해 120, 연사 0.08, 탄약 20, 전속성)

## 6. 세션 루프 설계

### 5분 세션 (캐주얼)
```
타이틀 → 스테이지 1 선택 → 5분 플레이 → 클리어
  → 레벨업 1회 + 무기 1개 획득 + 업적 1개
  → "다음 스테이지 해금!" → 한 판 더
```

### 15분 세션 (코어)
```
웨이브 서바이벌 → W15 도달 → 사망
  → "최고 기록까지 3웨이브" → 재도전
  → W18 → 합성 재료 발견 → "한 판만 더"
```

### 30분 세션 (하드코어)
```
보스 러시 → 3보스에서 사망 → 재도전
  → 5보스 클리어 → 점수 갱신
  → 일일 도전 확인 → "오늘은 근접만?" → 도전
```
