# 도파민 설계 — 뇌를 해킹하는 게임 디자인

> **"도파민은 보상이 아니라 '예측 오차'에서 나온다.
> 예상보다 좋은 일이 생길 때 뇌가 미친다."**

---

## 1. 즉각적 피드백 (Juice) — 0.1초 안에 터져라

### 킬 피드백 5중주
적을 잡으면 **동시에 5가지**가 터진다:

| 채널 | 효과 | 타이밍 |
|------|------|--------|
| **시각** | 폭발 파티클 16개 + 충격파 링 + 흰 플래시 | 0ms |
| **숫자** | "+150" 플로팅 텍스트 (콤보 시 크기 증가) | 0ms |
| **사운드** | 적 사망음 + 화음 레이어 + 리버브 | 0ms |
| **콤보** | "7 HIT" 카운터 펄스 + 스타일랭크 갱신 | 50ms |
| **전리품** | 12% 확률로 무기 crate 드롭 (반짝임) | 200ms |

### 크리티컬 히트 (예측 오차 #1)
- 일반 히트: 흰 숫자, 작은 스파크
- **크리티컬**: 주황색 "CRIT 247" 큰 글씨 + 화면 플래시 + 전용 사운드 + 충격파
- **약점 크리티컬**: 금색 "WEAKPOINT!" + 1.85배 + 별도 사운드
- 크리티컬 확률: 기본 6% → 레벨업/콤보/오버드라이브로 최대 47%
  → **"이번엔 크리 뜰까?"** 가 매 발마다 도박

### 엘리트 적 (예측 오차 #2)
- 일반 적 사이에 **갑자기** 빨간/파란/보라 오라를 두른 엘리트 등장
- "BERSERK" / "ARMORED" / "RAPID" / "VAMP" 라벨
- 처치 시: **2.4배 점수** + 전용 사운드 + "ELITE BREAK!" 토스트
- 등장 확률: 5.5% + 스테이지×1.8% + 레벨×0.2%
  → **"이번엔 엘리트 나올까?"** 가 매 스폰마다 도박

## 2. 가변 보상 (Variable Ratio Reinforcement)

### 슬롯머신 드롭 테이블
적 처치 시 드롭은 **랜덤**이다. 이 랜덤성이 중독의 핵심:

```
일반 적 처치:
  12% → 무기 crate (HMG? 로켓? 아크? 레일건?)
  6%  → 수류탄 ×3
  0%  → 체력 (일반 적에서는 안 나옴)

엘리트 처치:
  25% → 무기 crate
  15% → 수류탄 ×5
  10% → 체력 +40

미니보스/보스 처치:
  100% → 무기 crate (고등급)
  50% → 체력 +60
  30% → 수류탄 ×10
```

**"이번엔 레일건 나오면 대박인데"** → 매 킬이 슬롯머신

### 무기 합성 가챠
- HMG Lv3 + Laser Lv3 = **플라즈마 스톰** (확정)
- 하지만 **어떤 조합을 먼저 만드느냐**는 드롭 운에 달림
- "로켓이 먼저 Lv3이 됐는데, 아크가 안 나와..."
  → **수집 욕구 + 완성 욕구** 동시 자극

### 웨이브 서바이벌 보급
- 매 5웨이브마다 보급품 투하
- **무기가 랜덤** → "이번엔 뭐가 나올까?"
- 간혹 **레일건**이 나오면 → 도파민 폭발

## 3. 진행 바 중독 (Progress Bar Addiction)

### 항상 보이는 진행 바
화면에 **최소 3개의 진행 바**가 항상 표시된다:

```
┌─────────────────────────────────────────┐
│ [HP ████████░░] 80/100                  │  ← 체력 바
│ [FURY ██████░░░░] 62%  → OVERDRIVE!     │  ← 오버드라이브 게이지
│ [XP ████░░░░░░] LV.12  340/890         │  ← 경험치 바
│ [WAVE 7]  남은 적: ████░░ 12/18        │  ← 웨이브 진행 (서바이벌)
└─────────────────────────────────────────┘
```

**심리학**: 진행 바가 70%를 넘으면 "끝까지 채워야 해"라는 강박이 생긴다.
- XP 바가 90% → "한 마리만 더 잡으면 레벨업인데"
- Fury가 95% → "한 대만 더 때리면 오버드라이브인데"
- 웨이브 적 2마리 남음 → "2마리만 더"

### 레벨업 세레모니
레벨업 순간 **0.5초간 게임이 멈추고**:
1. 녹색 충격파 + 화면 플래시
2. "LEVEL UP 13" 금색 큰 글씨
3. 상승 스탯 나열: "화력+10 · 체력+5 · 치명타+0.8%"
4. 전용 화음 사운드 (7음계 상승)
5. **새 해금 알림**: "2단 점프 해금!" / "불릿 타임 해금!"

## 4. 콤보 시스템 — "끊기면 아깝다"

### 콤보 심리
- 7콤보: C랭크 → "B까지 9콤보 남았어"
- 16콤보: B랭크 → "A까지 12콤보, 한 웨이브면 되겠다"
- 28콤보: A랭크 → "S까지 17콤보... 여기서 죽으면 안 돼"
- 45콤보: S랭크 → **"S 유지해야 해, 절대 끊기면 안 돼"**

### 콤보 유지 메커니즘
- 콤보 타이머: 킬 시 2.3초, 히트 시 1.65초
- **콤보가 0이 되기 직전**에 적을 잡으면 → "CLOSE CALL!" 보너스
- 콤보 ×8마다 점수 배율 증가 (×1.25, ×1.50, ×1.75, ×2.00...)
- **S랭크 유지 중**: 화면 금색 비네트 + 전용 BGM 레이어 추가

### 콤보 끊김의 고통
- 30콤보에서 피격 → 콤보 리셋
- "아..." 소리가 절로 나옴
- **이 고통이 "다음엔 안 끊겨야지" → 재도전 동기**

## 5. 위험/보상 (Risk/Reward)

### 근접 처치 (EXECUTION)
- 적 HP 22% 이하에서 근접(L) → **즉사 + fury +20**
- 하지만 근접 = **피격 위험**
- "안전하게 쏠까, 리스크 감수하고 근접으로 끝낼까?"

### 탈것 탈취
- tank를 **근접으로** 잡아야 탈취 가능
- tank는 접근하면 포격 → **접근 자체가 리스크**
- 성공 시 30초간 무적 전차 → **보상이 미침**

### 크로노 브레이크
- 불릿 타임: fury 50 소비 → **오버드라이브를 포기해야 함**
- "오버드라이브(7초 화력 138%) vs 불릿 타임(3초 회피)"
  → **상황 판단의 재미**

### 보스 러시
- 5보스 연속, 체력 70%로 시작
- "1보스에서 오버드라이브 쓸까, 3보스를 위해 아낄까?"
  → **자원 관리의 깊이**

## 6. 발견의 기쁨 (Discovery)

### 숨겨진 요소
- 스테이지 특정 위치에서 **벽을 쏘면** 숨겨진 무기 (레일건 확정)
- 드럼통 뒤에 **숨겨진 포로** (일반 포로보다 2배 점수)
- 보스 특정 패턴을 **근접으로 반사**하면 → "PARRY!" + 3배 피해
- 웨이브 13, 17, 23에서 **특정 조건** 충족 → 히든 웨이브

### 합성 발견
- 첫 합성 성공 시: "⚡ NEW WEAPON DISCOVERED: PLASMA STORM"
- 합성 도감: 4종 중 몇 개 발견했는지 표시
- **전부 발견하면**: 히든 합성 "OMEGA BLASTER" 해금

### 업적 발견
- 업적 목록에 **???** 로 숨겨진 업적 5종
- 조건 충족 시에만 이름 공개
- "이 업적은 어떻게 깨는 거지?" → 커뮤니티 토론 유발

## 8. 일일 도전 (Daily Challenge)

### 날짜 기반 시드
```javascript
var dailySeed = new Date().toISOString().slice(0, 10); // "2026-07-24"
var rng = makeRng('daily:' + dailySeed);
```

### 매일 다른 조건 (예시)
| 날짜 | 조건 | 설명 |
|------|------|------|
| 월 | 권총만 | 보조 무기/합성 금지 |
| 화 | 대시 금지 | 이동으로만 회피 |
| 수 | 적 HP ×3 | 탱키하지만 점수 ×2 |
| 목 | 거울 모드 | 좌우 반전 |
| 금 | 근접만 | 총기 사용 금지 |
| 토 | 원코인 | 목숨 1개 |
| 일 | ALL RANDOM | 무기/적/보스 랜덤 |

### 일일 보상
- 클리어 시: **전용 업적** + XP ×2
- 7일 연속: **히든 무기 스킨** (색상 변경)
- 월간 개근: **전용 칭호**

## 9. 사운드 도파민

### 킬 사운드 계층
```
일반 킬: "퍽" (단순)
크리티컬: "퍽" + "챙!" (화음 추가)
엘리트 킬: "퍽" + "챙!" + "우웅~" (저음 리버브)
보스 킬: "콰과광" + 팬파레 7음계 + 리버브 1.2초
```

### 콤보 사운드 에스컬레이션
- 1-6콤보: 기본 킬 사운드
- 7-15콤보: + 하이햇 리듬
- 16-27콤보: + 베이스 라인
- 28-44콤보: + 아르페지오
- 45+콤보: **전체 BGM 레이어 + 전용 멜로디**

### 레벨업 사운드
```
C-E-G-C-E-G-C (상승 7음계)
+ 마지막 음에 리버브 1.5초
+ "짠!" 효과음 (금속성)
```

## 10. "한 판만 더" 트리거

### 세션 끝에서
- 웨이브 서바이벌 사망: "WAVE 18 도달 · 최고 기록까지 3웨이브"
  → **"3웨이브면 되는데..."**
- 스테이지 클리어: "다음 스테이지 해금! 새 적: 저격수"
  → **"새 적 구경해야지"**
- 보스 러시 3보스에서 사망: "4보스까지 1번 남았었음"
  → **"이번엔 4보스까지 갈 수 있어"**

### 시작할 때
- 타이틀 화면: "어제 기록: WAVE 18 · 점수 84,200"
  → **"오늘은 20 넘긴다"**
- 일일 도전: "오늘의 도전: 권총만 · 보상: XP ×2"
  → **"이건 해야 해"**

### 3분 룰
- **첫 3분 안에**: 레벨업 1회 + 무기 획득 1회 + 10콤보 1회
- 이 3가지가 **반드시** 일어나도록 스폰/드롭 밸런스 설계
- 3분 안에 도파민 3회 → **계속하게 됨**
