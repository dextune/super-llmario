## 추가로 넣으면 개쩌는 것들

### 🎪 공중 콤보
↑+L = 어퍼컷. 적이 공중으로 뜬다.
공중에서 사격 → "AIR COMBO ×3" → 근접 → "SKY BREAK!"
Devil May Cry가 브라우저에서 된다.

### 🧬 변이 보스
보스 잡았는데 15% 확률로 **다시 일어남**.
"MUTANT" 빨간 오라, 속도 2배, 새 패턴.
점수 3배. "아 씨발 또?" → 이기면 도파민 10배.

### 👽 히든 스테이지: AREA 51
업적 25개 모으면 해금.
외계인 적, 중력 이상(점프 1.5배), 보스 "THE ARCHITECT".
"이 스테이지가 있었다고?"

### 📅 일일 도전
매일 다른 조건. 월요일: 권총만. 금요일: 근접만.
"오늘은 거울 모드(좌우반전)?" → 이건 해야 해.
7일 연속 = 히든 스킨.

### 🏆 보스 챌린지
보스별 개별 타임어택.
"ASSAULT TANK 30초 컷" → SSS 랭크.
5보스 전부 SSS = "GODLIKE" 칭호.

### 💀 나이트메어 모드
스테이지 클리어 후 해금.
적 HP ×3, 속도 ×1.5, 엘리트 확률 40%.
클리어 시 전용 업적 + "NIGHTMARE CLEARED" 금색 텍스트.

### 🎰 랜덤 웨이브
웨이브 서바이벌에서 13/17/23웨이브는 **랜덤**.
적 구성, 수, 배율이 매번 다름.
"이번엔 뭐가 나올지 모른다" = 긴장감.

### 🔥 피버 모드
50콤보 달성 시 10초간 **피버**.
- 모든 공격 크리티컬
- 점수 ×5
- 화면 금색 + 전용 BGM
- 적 처치 시 폭발 3배
"50콤보 유지해야 해" → 손에 땀.

### 🎵 리듬 사격
특정 웨이브에서 BGM 비트에 맞춰 사격하면 **비트 샷**.
- 비트 적중: 피해 ×2 + 전용 사운드
- 8연속 비트: "PERFECT RHYTHM!" 보너스
"총을 음악 게임처럼 쏜다."

## 비밀/이스터에그

### 코나미 커맨드
타이틀에서 ↑↑↓↓←→←→BA
→ "GOD MODE" 30초 (무적 + 원샷원킬)
→ 전용 사운드 + 화면 레인보우

### 드럼통 농구
드럼통을 공중에서 터뜨리면 **농구 골** 이펙트.
"SWISH! +100" 토스트.

### 보스 도발
보스 앞에서 S키 3번 = **도발**.
보스 분노 (속도 +50%, 공격 +50%)
하지만 처치 시 점수 ×2.
"도발하고 이기면 2배" = 리스크/리워드.

### 숨겨진 포로
스테이지별 특정 위치에서 벽을 쏘면 **숨겨진 포로**.
일반 포로 500점 → 숨겨진 포로 2000점.
"여기 포로가 있었어?"

### 제너럴의 검
SUPREME COMMANDER를 **근접만으로** 처치.
→ "제너럴의 검" 획득 (영구 근접 피해 ×2)
→ 전용 업적 + 전용 사운드

## "한 판만 더" 심리 설계

### 3분 룰
첫 3분 안에 **반드시**:
- 레벨업 1회
- 무기 획득 1회
- 10콤보 1회
- 드럼통 폭발 1회

→ 3분 안에 도파민 4회 → 뇌가 "이 게임 재밌다" 판단

### 끝에서 2%
- XP 바 98% → "한 마리만 더"
- 웨이브 적 1마리 남음 → "1마리만"
- 보스 HP 2% → "2%면 죽는데"
- 합성 재료 1개 부족 → "하나만 더 나오면"

### 죽을 때 보여주는 것
```
WAVE 18 도달
최고 기록: WAVE 21
"3웨이브면 신기록인데..."

[다시 하기]  [타이틀로]
```
→ "3웨이브"가 뇌를 자극

### 이길 때 보여주는 것
```
WAVE 25 CLEAR!
점수: 128,400 (신기록!)
처치: 187체 | 최대 콤보: 43 | 크리티컬: 31회

[다음 웨이브 도전]  [타이틀로]
```
→ "신기록" + "다음 웨이브" = 무한 루프

## 사운드 도파민

### 킬 사운드 계층
```
일반:    "퍽"
크리티컬: "퍽" + "챙!"
엘리트:  "퍽" + "챙!" + "우웅~"
보스:    "콰과광" + 팬파레 7음계 + 리버브 1.2초
합성:    "찌이이잉—" + 폭발 + 화음
탈취:    엔진음 + "HIJACK!" 음성
크로노:  시계 태엽 + 역재생 + "틱...틱...틱..."
```

### 콤보 BGM 에스컬레이션
- 1-6: 기본 BGM
- 7-15: + 하이햇
- 16-27: + 베이스
- 28-44: + 아르페지오
- 45+: **전체 오케스트라 + 전용 멜로디**

### 피버 모드
- 50콤보: BGM이 **완전히 바뀜** (고속 템포 + 일렉 기타)
- 10초 후: 원래 BGM으로 돌아오며 "아..." 소리

## 접근성 (재미의 전제조건)

- **색맹 모드**: 적/아군 색상 구분 (빨강→주황, 파랑→노랑)
- **화면 흔들림 끄기**: 이미 구현됨
- **키 리매핑**: 모든 키 변경 가능
- **난이도 조절**: 웨이브 서바이벌에서 "이지/노말/하드" 선택
  - 이지: 적 HP ×0.5, 속도 ×0.8
  - 하드: 적 HP ×2, 속도 ×1.3, 엘리트 ×2
