# METAL STRIKE: OVERDRIVE — 게임 비전

> **"30초 안에 도파민, 3분 안에 중독, 30시간 안에 마스터"**

## 핵심 판타지

**"나는 멈추지 않는 전쟁 기계다."**

플레이어는 한 명의 병사로 시작하지만, 레벨이 오르고 무기를 합성하고 전차를 탈취하면서
**걷는 재앙**이 된다. 화면을 가득 채우는 폭발, 연쇄 번개, 시간 조작, 보스 러시.
Metal Slug의 손맛 + Diablo의 성장 + Devil May Cry의 스타일 + Hades의 중독성.

## 3대 설계 원칙

### 1. 매 순간이 하이라이트 (Highlight Reel Design)
- 평범한 순간이 없어야 한다
- 적 1마리를 잡아도 **파티클 + 사운드 + 숫자 + 진동**이 터진다
- 10콤보, 30콤보, 50콤보마다 **화면이 달라진다**
- 보스를 잡으면 **5연속 폭발 + 슬로모 + 팬파레**

### 2. "한 판만 더" 루프 (One More Run Loop)
- 웨이브 서바이벌: "이번엔 15웨이브까지 갔는데, 다음엔 20까지 갈 수 있을 것 같아"
- 무기 합성: "HMG+Laser 합성하면 뭐가 나올까?"
- 크로노 브레이크: "죽을 뻔했는데 시간 돌리니까 살았어, 이건 무조건 클리어각"
- 일일 도전: "오늘은 권총만 모드? 이건 해야 해"

### 3. 성장 체감 100% (Felt Progression)
- 레벨 1과 레벨 50은 **완전히 다른 게임**
- 총알이 커지고, 번개가 5마리까지 튀고, 전차를 맨손으로 잡는다
- 무기 Lv.3 + Lv.3 = **합성 무기** — 시각적으로 완전히 다른 이펙트
- 업적 12개 → 30개 → 50개로 수집 욕구 자극

| 유형 | 니즈 | Metal Strike가 주는 것 |
|------|------|----------------------|
| 아케이드 매니아 | 즉각적 손맛 | 대시/근접/콤보/스타일랭크 |
| 수집가 | 완성 욕구 | 업적 50종, 무기 합성 도감, 보스 도감 |
| 도전자 | 한계 돌파 | 보스 러시, 웨이브 서바이벌, 일일 도전 |
| 성장형 플레이어 | 강해지는 느낌 | 레벨 50, 무기 Lv.3, 합성, 크로노 해금 |
| 캐주얼 | 짧은 세션 | 1스테이지 5분, 웨이브 10분 |

## 기술 스택

- **순수 JavaScript + Canvas 2D** (외부 의존성 0)
- WebAudio 칩튠 (리버브/딜레이/화음)
- 16×16 픽셀아트 (3× 스케일)
- localStorage 세이브
- 소스 어셈블리 + SHA-256 무결성 검증
- Overdrive 패치 시스템 (런타임 소스 변환)
