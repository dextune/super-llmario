# 핵심 메카닉 — 4가지 미친 기능

## 1. 🌊 웨이브 서바이벌 (Wave Survival)

### 컨셉
> "끝이 없다. 너가 멈출 때까지."

5스테이지 클리어 후 해금되는 **무한 전투 모드**.
arena(1600px)에서 끝없이 몰려오는 적을 상대한다.
웨이브가 오를수록 적의 HP/속도/수가 증가하고, 10웨이브마다 미니보스, 25웨이브마다 보스가 나온다.

### 웨이브 테이블

| 웨이브 | 적 구성 | HP 배율 | 엘리트 확률 | 특수 |
|--------|---------|---------|------------|------|
| 1-3 | grunt ×6, rifleman ×2 | ×1.0 | 5% | 튜토리얼 |
| 4-5 | grunt ×8, rifleman ×4, shield ×2 | ×1.1 | 8% | |
| 6-7 | +bazooka ×2, grenadier ×2 | ×1.2 | 10% | |
| 8-9 | +sniper ×2, turret ×1 | ×1.3 | 12% | |
| **10** | **MINIBOSS: TANK** + grunt ×6 | ×1.4 | 15% | 🎵 보스 음악 |
| 11-13 | +drone ×4, shield ×3 | ×1.5 | 15% | 드론 첫 등장 |
| 14-15 | +bazooka ×3, sniper ×3 | ×1.6 | 18% | |
| 16-17 | +turret ×2, grenadier ×3 | ×1.7 | 20% | |
| 18-19 | +drone ×6, rifleman ×5 | ×1.8 | 22% | |
| **20** | **MINIBOSS: CHOPPER** + drone ×4 | ×1.9 | 25% | 🎵 보스 음악 |
| 21-23 | 전 적종 혼합 ×20 | ×2.0 | 25% | 엘리트 러시 |
| 24 | 전 적종 ×25 + turret ×3 | ×2.1 | 28% | |
| **25** | **BOSS (랜덤)** + grunt ×8 | ×2.2 | 30% | 💀 보스 음악 |
| 26+ | 루프, HP ×1.5 추가 | ×(2.2+0.5n) | 30%+ | ∞ |

### 웨이브 흐름
```
[웨이브 시작] → "WAVE 7" 토스트 + 경고음
  → 적 스폰 (arena 좌우 + 공중)
  → 전원 처치
[웨이브 클리어] → "WAVE CLEAR! +500" + 5초 브레이크
  → 보급품 투하 (체력/탄약/수류탄/랜덤 무기)
  → 카운트다운 5...4...3...2...1
[다음 웨이브]
```

### 점수 시스템
- 킬 점수 × 웨이브 배율 (wave × 0.5 + 1)
- 웨이브 클리어 보너스: wave × 200
- 미니보스: ×3, 보스: ×5
- **무피해 웨이브**: ×2 추가 보너스
- 최고 기록: localStorage 저장 (`bestSurvivalWave`, `bestSurvivalScore`)

### UI
- 화면 상단 중앙: **WAVE 12** (큰 글씨, 펄스)
- 상단 좌측: 남은 적 수 ( skull 아이콘 × n )
- 상단 우측: 서바이벌 점수
- 브레이크 중: 화면 중앙에 카운트다운 + "다음 웨이브: shield ×3, bazooka ×2" 미리보기

## 2. ⚔️ 무기 합성 (Weapon Fusion)

### 컨셉
> "두 개의 최강을 하나로. 그 순간 화면이 하얘진다."

무기 2개를 동시에 소지하고, **둘 다 Lv.3**일 때 합성하면
**완전히 새로운 무기**가 탄생한다. 합성 순간 화면 플래시 + 전용 사운드 + 전용 이펙트.

### 2슬롯 시스템
- **주 무기 (1번 슬롯)**: 현재 사용 중
- **보조 무기 (2번 슬롯)**: Q키로 스왑
- 무기 픽업 시: 현재와 같으면 **강화**, 다르면 **보조 슬롯에 저장**
- HUD에 `[H] HMG ★3 | [L] Laser ★2` 형태로 2슬롯 표시

### 합성 레시피

| 조합 | 결과 | 속성 | 비주얼 |
|------|------|------|--------|
| HMG ★3 + Laser ★3 | **플라즈마 스톰** | dmg 35, fireRate 0.04, 관통, 40발 | 보라색 플라즈마 + 잔상 trail |
| Rocket ★3 + Arc ★3 | **썬더 크래시** | dmg 90, fireRate 0.6, 폭발+체인, 15발 | 청백색 폭발 + 번개 동시 |
| Railgun ★3 + HMG ★3 | **개틀링 레일** | dmg 50, fireRate 0.15, 관통, 30발 | 금색 고속 탄환 + 궤적 |
| 기타 조합 | **퓨전 블래스터** | dmg 45, fireRate 0.12, 35발 | 백색 에너지 볼 |

### 합성 연출 (도파민 폭발 구간)
1. Q+L 입력 → 0.5초간 **화면 정지** (hitStop)
2. 두 무기 아이콘이 화면 중앙으로 날아옴
3. **충돌 순간**: 화면 백플래시 + 충격파 + 전용 화음 사운드
4. 새 무기 이름이 **금색 큰 글씨**로 나타남: "⚡ PLASMA STORM ⚡"
5. 플레이어 주변에 **에너지 오라** 3초간 유지
6. 첫 발사: 일반 무기보다 **3배 큰 총알 + 화면 미세 진동**

### 합성 무기 만료
- ammo 소진 시: "FUSION EXPIRED" 토스트 + pistol 전환
- 합성 무기는 **재합성 불가** (한 번 쓰면 끝) → "아껴 쓸까, 지금 쓸까" 딜레마

## 3. 🚀 탈것 탈취 (Vehicle Hijack)

### 컨셉
> "적의 무기가 내 무기가 된다. 전차를 맨손으로 잡아라."

tank/chopper를 **근접(L)으로 처치**하면 30% 확률로 **탈것 픽업**이 드롭된다.
주으면 30초(전차) 또는 15초(제트팩)간 **압도적 화력**을 얻는다.

### 전차 모드 (Tank Ride)
- **지속**: 30초
- **이동**: 속도 120 (느리지만), 점프 불가
- **방어**: 소형 총알 무효화 (rifleman, grunt, drone 공격 무시)
- **공격**:
  - J: 주포 발사 (dmg 40, fireRate 0.5, 폭발 반경 60)
  - 자동 MG: 3발 스프레이 (dmg 8, fireRate 0.1)
- **접촉 피해**: 적을 밟으면 dmg 50 + 넉백
- **체력**: vehicle HP 200 (bazooka/rocket/turret 공격에 감소)
- **만료**: 폭발 + 플레이어 이젝트 (2초 무적)

### 제트팩 모드 (Jetpack)
- **지속**: 15초
- **비행**: SPACE 홀드 = 상승, 놓으면 하강 (중력 0.3배)
- **이동**: 공중 자유 이동, 속도 250
- **공격**: J = 전방 연사 (dmg 12, fireRate 0.08)
- **만료**: 낙하산 (3초간 느린 하강, 무적)

### 탈취 연출
1. 근접으로 tank/chopper 처치 시
2. 폭발 대신 **"HIJACK!"** 텍스트 + 녹색 플래시
3. 탈것 픽업 드롭 (반짝이는 아이콘)
4. 픽업 시: **전환 애니메이션** (0.3초, 플레이어 스프라이트 → 탈것 스프라이트)
5. 전용 사운드: 전차 = 엔진음, 제트팩 = 제트음

### 전략적 깊이
- 전차는 **느리지만 무적** → 보스전에서 탱킹
- 제트팩은 **빠르지만 약함** → 드론/건십 상대 + 회피
- "근접으로 잡아야 한다"는 조건 → **리스크/리워드** (근접 = 피격 위험)
- 엘리트 tank는 탈취 불가 → "아, 이건 못 타"

## 4. ⏰ 크로노 브레이크 (Chrono Break)

### 컨셉
> "죽음은 끝이 아니다. 시간을 되돌려라."

두 가지 시간 조작 능력:
- **자동 리와인드**: 죽는 순간 1초 전으로 자동 복귀 (스테이지당 1회)
- **불릿 타임**: 적 총알이 0.2배속으로 느려지는 3초 (fury 50 소비)

### 자동 리와인드 (Auto Rewind)
- **발동 조건**: player.hp <= 0 AND `world.chronoUsed === false` AND 스냅샷 60프레임 이상
- **효과**:
  1. 화면 **청색 정지** (0.5초)
  2. "⏰ CHRONO BREAK!" 큰 글씨
  3. 1초 전 위치/체력/무기 복원
  4. 2.5초 무적
  5. 리와인드 이펙트: **역재생 파티클** (파티클이 역방향으로 수렴)
- **제한**: 스테이지당 1회, 보스 러시에서는 라운드당 1회
- **HUD**: "⏰ ×1" (사용 가능) / "⏰ ×0" (사용 완료, 회색)

### 불릿 타임 (Bullet Time)
- **발동**: C키 (fury 50 이상 필요)
- **지속**: 3초
- **효과**:
  - 적 총알 속도 ×0.2 (플레이어 총알은 정상)
  - 화면 **채도 감소** (반투명 청색 오버레이)
  - 적 총알에 **motion trail** (잔상 3프레임)
  - 플레이어 주변 **시간 왜곡 링**
- **전략**: 보스의 탄막 패턴을 **걸어서 회피** → 반격
- **사운드**: 저음 슬로우 + 시계 틱톡

### 스냅샷 시스템 (기술)
```
world.chronoBuf = []  // 링 버퍼, 최대 180프레임

// 매 프레임:
world.chronoBuf.push({
  x: player.x, y: player.y, hp: player.hp,
  vx: player.vx, vy: player.vy,
  weapon: player.weapon, ammo: player.weaponAmmo
});
if (world.chronoBuf.length > 180) world.chronoBuf.shift();

// 리와인드:
var snap = world.chronoBuf[Math.max(0, world.chronoBuf.length - 60)];
player.x = snap.x; player.y = snap.y; player.hp = snap.hp;
// ... 복원
```

### 크로노 브레이크 연출 (도파민 구간)
1. 사망 판정 → 화면 **0.5초 정지**
2. 화면이 **청색으로 물들며** 시계 효과음
3. 플레이어 잔상이 **역방향으로 이동** (스와프 애니메이션)
4. "CHRONO BREAK" 텍스트 + 충격파
5. 2.5초 무적 + **시간의 모래** 파티클 (금색 반짝임)
6. BGM 잠시 정지 → 재개

## 기능 간 시너지

```
[웨이브 서바이벌 W15]
  → 드론 6마리 습격
  → 아크 라이플로 4체인 ⚡
  → 남은 2마리는 대시로 회피 💨
  → 웨이브 클리어, 보급에서 HMG 드롭
  → HMG Lv3 + Laser Lv3 → ⚡ PLASMA STORM 합성!
  → W20 건십 등장 → 근접으로 처치 → 🚁 제트팩 탈취!
  → 15초간 공중에서 플라즈마 스톰 난사
  → W25 보스전 → 탄막 패턴 → ⏰ 불릿 타임으로 회피
  → 보스 HP 10% → 근접 EXECUTION 💀
  → "WAVE 25 CLEAR! 점수: 128,400"
```
